# Apotek Dokter Aqa & Fitria PWA

Sistem Manajemen Inventaris Obat untuk Apotek Dokter Aqa & Fitria.

## Fitur
- Dashboard ringkasan stok
- Inventaris obat dengan status kritis/warning/aman
- Transaksi stok masuk dan keluar
- Rekap laporan
- Export PDF/HTML
- Real-time sync dengan Supabase
